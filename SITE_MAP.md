## Key Navigation Flows

### Primary User Journeys

**Journey 1: Service Discovery → Enquiry**
```
Home → Practice Areas → Services (detailed view) → Contact Form
```

**Journey 2: Educational → Trust → Enquiry**
```
Home → Resources → Insights/Tools → Contact Form
```

**Journey 3: Direct Contact**
```
Any Page → WhatsApp Button / Phone / Contact Page
```

**Journey 4: Tool Usage → Assistance**
```
Tools → Calculator → Result + CTA → Contact Form (pre-selected service)
```

## Contact Touchpoints (Multiple Conversion Paths)

1. **Header**: "Contact" navigation item
2. **Home CTA**: "Send an Enquiry" button
3. **Practice Areas**: "View all services" → Contact links
4. **Services**: "Send an enquiry for this service" (per service)
5. **Tools**: "Get professional assistance" (per calculator)
6. **Resources**: Article CTAs to contact
7. **Footer**: Contact details and links
8. **WhatsApp**: Floating button on all pages
9. **Bottom CTA**: "Get in Touch" section on home

**Total Contact Opportunities**: 15+ touchpoints across the site

## Responsive Breakpoints

- **Mobile**: 320px - 767px (hamburger menu, stacked layout)
- **Tablet**: 768px - 1023px (2-column where appropriate)
- **Desktop**: 1024px - 1279px (full navigation, multi-column)
- **Large**: 1280px+ (max-width containers, optimal reading width)

## Accessibility Features

- Semantic HTML5 structure
- ARIA labels on interactive elements
- Keyboard navigation support
- Focus indicators
- Alt text for icons (aria-hidden where decorative)
- Proper heading hierarchy (h1 → h6)
- Color contrast compliance
- Skip navigation links (can be added)
- Screen reader friendly
